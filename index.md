# bank_importer

A personal bank-transaction importer. It reads bank exports (CSV, XLS, XLSX, PDF) or pulls transactions directly from connected institutions via [Plaid](https://plaid.com), normalizes them, and appends the result to a private Google Sheet for personal accounting and tax record-keeping.

The application is a single-operator local utility — it runs on the operator's own workstation, binds only to the loopback interface (`127.0.0.1`), and is not a multi-user product. There is no signup, no hosted service, and no consumer-facing surface.
